# BasketPulse Helper

BasketPulse Helper 是一個非官方 Chrome 擴充功能，用於強化 BasketPulse 的球員訓練頁、籃球學校頁與比賽頁瀏覽體驗。

本工具目前提供兩大功能：

1. **球員訓練輔助**
   - 在球員技能頁與籃球學校頁顯示潛力素質、最新訓練結果。
   - 支援將球員名單匯出成 PNG 圖片。
   - 支援主要球員、出借球員、全部球員與自選球員匯出。

2. **比賽數據強化**
   - 在比賽 description 頁載入 play-by-play。
   - 自動計算進階 Box Score 數據。
   - 將 DFGA、DTS%、失分、TS%、位置時間、籃板競爭、非官方 EFF 等資料補充到官方 Box Score。
   - 支援強化模式下的自訂排序。

> 本工具不是 BasketPulse 官方工具，與 BasketPulse 官方無關。

## 功能總覽

## 一、球員訓練輔助功能

### 1. 潛力素質顯示

在球員技能頁與籃球學校頁的球員表格下方新增「潛力素質」資料列。

資料來源：

```text
https://www.basketpulse.com/tw/Training/overview
```

支援欄位：

```text
健康
彈跳
速度
韌性
近投
中投
三分
籃板
阻攻
防守智商
運球
傳球
進攻智商
經驗
```

---

### 2. 訓練結果顯示

在球員表格下方新增「訓練結果」資料列，顯示最新訓練提升數值。

範例：

```text
+1
+2
-1
```

---

### 3. 訓練總覽快取

BasketPulse Helper 會在瀏覽器本機使用 `chrome.storage.local` 暫存訓練總覽 HTML。

目的：

- 減少重複讀取訓練總覽。
- 加快載入速度。
- 降低 BasketPulse 伺服器負擔。

目前快取時間約為：

```text
10 分鐘
```

如果想重新取得最新資料，請點選工具列中的：

```text
更新資料
```

---

### 4. PNG 圖片匯出

支援將球員名單匯出成 PNG 圖片。

匯出範圍包含：

```text
主要球員
出借球員
全部球員
自選球員
```

若頁面沒有出借 / 租借球員區塊，則只會顯示：

```text
全部球員
自選球員
```

匯出檔名格式範例：

```text
BasketPulse_全部球員_2026-07-03_1340.png
```

---

### 5. 自選球員匯出

選擇：

```text
自選球員
```

後，頁面會顯示 checkbox，可自行勾選要匯出的球員。

若頁面有出借 / 租借球員區塊，快速選取會提供：

```text
全選
取消全選
只選主要
只選出借
```

若沒有出借 / 租借球員區塊，則提供：

```text
全選
取消全選
```

---

### 6. 顯示 / 隱藏資料列

球員訓練工具列提供：

```text
隱藏資料列
顯示資料列
```

可快速切換潛力素質列與訓練結果列的顯示狀態。

## 二、比賽數據強化功能

### 1. 支援頁面

請進入比賽 description 頁：

```text
https://www.basketpulse.com/tw/Match/{matchId}/description
```

頁面上方會出現 BasketPulse Helper 比賽工具列。

---

### 2. 強化數據顯示

點選：

```text
強化數據顯示
```

工具會執行以下流程：

1. 取得目前頁面的 Match ID。
2. 透過隱藏 iframe 載入 play-by-play 頁面。
3. 等待 play-by-play 事件完整渲染。
4. 解析 rendered DOM rows。
5. 將逐球資料轉換成 match data rows。
6. 根據內建比賽規則產生 box score rows。
7. 將進階數據補充到官方 Box Score。
8. 在官方 Box Score 下方產生強化分析區。

---

### 3. 官方 Box Score 主表補充欄位

強化後會在官方 Box Score 最右側新增：

```text
DFGA
DTS%
```

欄位說明：

| 欄位 | 說明 |
|---|---|
| DFGA | 該球員作為防守者時，對手在其防守下的出手次數 |
| DTS% | Defensive True Shooting 類似概念，用於估算該球員防守下的失分效率 |

---

### 4. 球員補充資料列

每位球員官方資料列下方會新增一列補充資料列。

補充列會垂直對齊官方欄位顯示：

| 對齊欄位 | 補充內容 |
|---|---|
| 分鐘 | 位置時間 |
| 得分 | 失分 |
| 2% | 近距離 / 中距離二分拆分 |
| 1% | TS% |
| 防守籃板 | 防守籃板競爭 |
| 進攻籃板 | 進攻籃板競爭 |
| 效率 | 非官方 EFF |

---

### 5. 位置時間顯示

位置時間只顯示有實際出場的位置，以節省版面。

範例：

```text
SF 18:35
SG 16:43 / SF 1:33
PF 7:10 / C 9:09
```

不會顯示無出場位置的 `0:00`。

例如不會顯示成：

```text
PG 0:00 / SG 0:00 / SF 18:35 / PF 0:00 / C 0:00
```

---

### 6. 失分

補充列會在官方「得分」欄下方顯示該球員作為防守者時，被對手取得的分數。

顯示範例：

```text
失 12
```

---

### 7. TS%

補充列會在官方「1%」欄下方顯示 TS%。

顯示範例：

```text
TS 64.3%
```

---

### 8. 非官方 EFF

非官方 EFF 由插件根據 play-by-play 計算，公式近似傳統效率值。

概念包含：

```text
得分
+ 總籃板
+ 助攻
+ 抄截
+ 封蓋
+ 命中數
- 出手數
+ 罰球命中
- 罰球出手
- 失誤
```

顯示位置會直接對齊官方效率欄位：

```html
<td name="ranking">
```

因此不依賴表頭 index，避免官方表格有 rowspan / colspan 時發生錯位。

---

### 9. 籃板競爭

補充列會在籃板欄位下方顯示籃板競爭資料。

範例：

```text
守 5/8 62.5%
攻 2/5 40.0%
```

其中：

- `守`：防守籃板競爭。
- `攻`：進攻籃板競爭。

---

### 10. 二分拆分

補充列會在官方 `2%` 欄位下方顯示近距離 / 中距離出手拆分。

範例：

```text
近 4/7｜中 2/5
```

## 三、底部強化分析區

官方 Box Score 下方會新增「強化數據補充」區塊。

目前包含三個分頁：

```text
出手機會品質
防守出手機會品質
傳球品質
```

---

### 1. 出手機會品質

統計球員在不同出手機會下的命中 / 出手。

分類包含：

```text
極差
糟糕
普通
良好
極佳
```

---

### 2. 防守出手機會品質

統計對手在該球員防守下，不同出手機會的命中 / 出手。

分類包含：

```text
對手極差
對手糟糕
對手普通
對手良好
對手極佳
```

---

### 3. 傳球品質

統計球員傳球品質與觸球率。

包含：

```text
糟糕的傳球
普通的傳球
安全的傳球
不錯的傳球
漂亮的妙傳
總傳球
觸球率
```

## 四、比賽頁自訂排序

強化模式下，工具列會啟用自訂排序功能。

### 1. 可排序項目

```text
總出場時間
各位置出場時間
得分
失分
TS%
防守籃板
防守籃板率
進攻籃板
進攻籃板率
總籃板
助攻
抄截
封蓋
效率
非官EFF
DFGA
DTS%
```

---

### 2. 排序方向

```text
高到低
低到高
```

---

### 3. 各位置出場時間排序

當排序項目選擇：

```text
各位置出場時間
```

會多顯示位置下拉選單：

```text
PG
SG
SF
PF
C
```

排序時會優先比較所選位置的時間，其餘位置再依內部順序作為次排序依據。

---

### 4. Row Pair 排序機制

強化後官方表格 DOM 會變成：

```text
球員官方列
球員補充列
球員官方列
球員補充列
```

自訂排序時，BasketPulse Helper 會把：

```text
官方球員列 + 補充資料列
```

視為同一筆資料一起移動。

這可以避免排序後補充列與球員列分離。

---

## 五、強化資料顯示 / 隱藏

比賽頁工具列提供：

```text
強化數據顯示
隱藏強化資料
顯示強化資料
```

用途：

- `強化數據顯示`：載入 play-by-play 並產生強化資料。
- `隱藏強化資料`：隱藏補充欄位、補充列與底部分析區。
- `顯示強化資料`：重新顯示已載入的強化資料，不重新載入 play-by-play。

## 安裝方式：開發者模式安裝

目前本工具可透過 Chrome 開發者模式安裝。

---

### 1. 下載專案

請從 GitHub 下載本專案 ZIP，或使用 Git clone。

```bash
git clone https://github.com/YOUR_USERNAME/basketpulse-helper.git
```

請將 `YOUR_USERNAME` 替換成實際 GitHub 帳號或專案所在位置。

---

### 2. 確認檔案結構

請確認資料夾內至少包含：

```text
basketpulse-helper/
├── manifest.json
├── README.md
├── THIRD_PARTY_LICENSES.txt
├── src/
│   ├── background/
│   │   └── background.js
│   ├── content/
│   │   ├── content.js
│   │   ├── content.css
│   │   ├── core/
│   │   ├── training/
│   │   └── match/
│   └── vendor/
│       └── html2canvas.min.js
```

---

### 3. 安裝到 Chrome

1. 開啟 Chrome。
2. 前往：

```text
chrome://extensions/
```

3. 右上角開啟「開發人員模式」。
4. 點選「載入未封裝項目」。
5. 選擇本專案資料夾：

```text
basketpulse-helper/
```

6. 安裝完成。

## 重要使用提醒

### 1. 切換球員頁或學校頁後，可能需要更新資料

BasketPulse Helper 會使用本機快取加快訓練總覽讀取速度。

因此在以下情況中，可能出現資料沒有正常顯示：

- 從球員技能頁切換到籃球學校頁。
- 從籃球學校頁切換回球員技能頁。
- BasketPulse 訓練總覽內容已更新。
- 頁面球員名單與快取中的資料不一致。

如果遇到上述狀況，請照以下順序操作：

```text
1. 點選「更新資料」
2. 再點選「載入潛力 / 訓練資料」
```

---

### 2. 比賽強化功能需要等待 play-by-play 載入

比賽強化功能會透過隱藏 iframe 載入 play-by-play 頁面，並等待頁面完整渲染。

因此點選「強化數據顯示」後，可能需要等待一段時間。

## 權限說明

本擴充功能使用以下權限：

---

### storage

用於在使用者瀏覽器本機儲存訓練總覽快取。

---

### host_permissions

```text
https://www.basketpulse.com/*
```

用途：

- 讀取目前登入帳號可查看的 BasketPulse 頁面。
- 抓取訓練總覽。
- 載入比賽 play-by-play。
- 在本機瀏覽器中解析資料。

---

## 隱私權說明

BasketPulse Helper：

- 不會蒐集個人資料。
- 不會傳送資料到開發者伺服器。
- 不會傳送資料到第三方伺服器。
- 只會在使用者瀏覽器本機處理頁面資料。
- 只會在使用者本機暫存訓練總覽 HTML，以加快讀取速度。

---

## 注意事項

- 本工具需要登入 BasketPulse 後使用。
- 本工具只會讀取目前登入帳號可查看的資料。
- 若 BasketPulse 網站結構更新，本工具可能需要同步更新。
- 若 BasketPulse 伺服器暫時回應 502 / 503 / 504，請稍後再試。
- 比賽強化功能需要等待 play-by-play 頁面完整渲染，請耐心等待。
- 本工具不是 BasketPulse 官方工具。
